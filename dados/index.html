Curso React - Dados de exemplo

Esta página contém um conjunto de dados de exemplo (arquivos JSON, imagens, etc.) para auxiliar no desenvolvimento de aplicações.

  • api/cardapio.json: JSON array com dados fictícios de um cardápio de lanchonete.
